Get event by id

Get the event by is id, return the event infos as well as the suffs, tasks and attendees linked to it

URL event/:eventId

eventId : id of the event
Method GET
URL Params Required: eventId : integer

example: eventId=23
Success response

Code : 200

Content : 
{
  "id": 1,
  "name": "Martini",
  "event": 1,
  "owner": 2039
}

Error response

Code : 404

Content : 
{
  "error": "No Stuff correspond to this id in the database"
}
